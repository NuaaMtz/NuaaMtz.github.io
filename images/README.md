# 图片目录

这个目录用于存放网站所需的图片文件。

## 需要的图片文件

### 1. profile.jpg
- **用途**: 个人照片
- **尺寸建议**: 至少 400x400 像素
- **格式**: JPG 或 PNG
- **要求**: 正方形或接近正方形，清晰的专业照片

### 2. background.jpg
- **用途**: 网站背景图片
- **尺寸建议**: 1920x1080 或更大
- **格式**: JPG
- **要求**: 学术风格，可以是：
  - 校园风景
  - 图书馆
  - 实验室
  - 简洁的几何图案
  - 或其他学术相关的背景

### 3. profile-placeholder.jpg (可选)
- **用途**: 当 profile.jpg 不存在时的占位图片
- **尺寸**: 400x400 像素
- **格式**: JPG 或 PNG

## 如何添加图片

1. 将你的个人照片命名为 `profile.jpg` 并放入此目录
2. 将背景图片命名为 `background.jpg` 并放入此目录
3. 使用 Git 提交：
   ```bash
   git add images/profile.jpg images/background.jpg
   git commit -m "添加个人照片和背景图片"
   git push
   ```

## 注意事项

- 图片文件大小建议控制在 2MB 以内，以确保网站加载速度
- 个人照片建议使用专业的证件照或半身照
- 背景图片建议使用较淡的颜色，以免影响文字可读性
