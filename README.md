# 个人主页

这是我的个人学习主页，包含个人介绍和文档下载功能。

## 功能特性

- 🎨 现代化的响应式设计
- 📱 移动端友好
- 📚 文档下载模块
- 🔗 社交媒体链接

## 文件结构

```
.
├── index.html          # 主页面
├── styles.css          # 样式文件
├── script.js           # JavaScript 功能
├── documents/          # 文档存储目录
│   └── README.md       # 文档目录说明
└── documents-list.json # 文档列表配置
```

## 如何添加文档

1. 将文档文件复制到 `documents/` 目录
2. 编辑 `documents-list.json`，添加新文档信息
3. 提交并推送到 GitHub

详细说明请查看 `documents/README.md`

## 本地开发

直接在浏览器中打开 `index.html` 即可查看效果。

## 部署

这个仓库配置为 GitHub Pages，推送到 `main` 分支后会自动部署。
